## Описание

Random Task Generator — это графическое приложение на Python (Tkinter), которое помогает пользователю получать случайные задачи из предопределённого списка, добавлять свои собственные задачи с указанием типа, сохранять историю в JSON-файл и фильтровать задачи по типу.

## Возможности

- Генерация случайной задачи из списка (учёба, спорт, работа)
- Добавление новых задач с валидацией (не пустая строка)
- Сохранение истории в JSON и загрузка при запуске
- Фильтрация истории по типу задачи
- Просмотр истории с временными метками
- Очистка истории
- Удобный графический интерфейс
